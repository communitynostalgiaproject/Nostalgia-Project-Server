Flags

Create a new flag

Authorizations:
googleAuth
Request Body schema: application/json
required
_id
string

Unique identifier for the Flag.

contentId
string

Identifier of the content being flagged.

contentType
string
Enum: "Experience" "Comment"

Type of content being flagged.

userId
string

Identifier of the user who flagged the content.

createdDate
string <date-time>

Date and time when the flag was created.

priority
string
Enum: "low" "medium" "high"

Priority level of the flag.

reason
string
Enum: "spam" "hate-speech" "misinformation" "other"

Reason for flagging the content.

userComment
string

Comments from the user who flagged the content.

moderatorComments
Array of strings

Comments from moderators on the flag.

resolved
boolean

Indicates whether the flag has been resolved.

contentDeleted
boolean

Indicates whether the content associated with the flag has been deleted.

resolvedBy
string

Identifier of the moderator who resolved the flag.

Responses

/flags
